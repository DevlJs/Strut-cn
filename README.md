### 开发/构建 ###

构建你自己的版本的需要Gruntv0.4.0＋。

1. 安装grunt `npm install -g grunt-cli`
2. 克隆Strut：`git clone https://github.com/biezhi/Strut-cn`
3. `cd Strut-cn`
4. 安装依赖：`npm install`
5. 运行：`grunt server`（这个服务监听在localhost:9000）

你可以使用 `grunt build` 用于构建到生产环境。
